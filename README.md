# Nivara Hotel Website

A responsive hotel website built with React that showcases rooms, wellness,
experiences, and events for a luxury eco-hotel concept.

## Live Site

https://thinyushwe.github.io/luxury-hotel-and-resort/

## Tech Stack

- React
- React Router
- CSS (custom, no framework)

## Features

- Responsive layout (mobile & desktop)
- Accessible semantic HTML structure
- Expandable content sections
- Image-led design with animations

## Accessibility

This project follows accessibility best practices including:

- Semantic landmarks
- Keyboard-accessible navigation
- Proper heading hierarchy
- Accessible buttons and links

## Known Limitations / Future Improvements

- Improve form validation and autocomplete
- Refactor hard-coded content into reusable data structures
- Further accessibility audits
